Simple Voice Recorder

A modern, clean, and privacy-first voice recorder for Android.

✨ Features

- High-quality audio recording
- Pause and resume recording
- Real-time recording timer
- Live waveform visualization
- Favorites
- Folder management
- Trash and restore
- Fast search
- Multiple audio formats
- Material 3 design
- Dark mode
- Offline-first
- No ads
- Open source

🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Room
- Coroutines
- Flow
- MVVM
