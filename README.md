Simple Voice Recorder  A modern, privacy-respecting voice recorder for Android — built with Kotlin, Jetpack Compose, and Clean Architecture.  ## Features  - Record in M4A, WAV, or FLAC - Pause / resume recording - Real-time waveform and amplitude display - Background recording with a foreground service - Playback with variable speed, A-B loop, and sleep timer - Folders, playlists, bookmarks, and full-text search - Trash with auto-restore window - App lock (PIN / biometric) - Local ZIP backup and restore - Dark mode, dynamic color, and 40+ languages  ## Tech stack  - 100% Kotlin - Jetpack Compose + Material 3 - MVVM + Clean Architecture (domain / data / ui / core) - Kotlin Coroutines + StateFlow - Room database - WorkManager (scheduled backup/cleanup)  ## Project structure  See ARCHITECTURE.md for the full module breakdown.  ## Building  bash git clone https://github.com/<your-username>/SimpleVoiceRecorder.git cd SimpleVoiceRecorder ./gradlew assembleDebug   The debug APK will be at app/build/outputs/apk/debug/app-debug.apk.  ## Contributing  See CONTRIBUTING.md.  ## License  Apache License 2.0 — see LICENSE.
